# MyUnsplash

Full stack unsplash clone

## Building and running on localhost

<ol>
<li>
First create .env file in backend folder like <strong><em>.env.template</em></strong> provided.</li>

<li>
Then install dependencies:

```sh
yarn install
```

</li>

<li>
To run backend:

```sh
yarn workspace @mu/backend start
```

</li>
<li>
To run frontend:

```sh
yarn workspace @mu/frontend start
```

</li>

</ol>
